# Timberland Regional Library: Anatomy of a Financial Crisis

**[View the interactive data visualization](https://ecwilsonaz.github.io/trl-crisis-charts/)**

A data-driven investigation into how Timberland Regional Library (TRL) leadership claimed financial health while fund balances collapsed — then laid off 61 frontline employees in March 2026.

## What This Shows

Eight exhibits built from public records tell the story:

- **Exhibit A** — Revenue vs. expenditures: the "scissors" that opened in 2023
- **Exhibit B** — Leadership quotes vs. what actually happened
- **Exhibit C** — Combined fund balance collapse (down $12.2M from peak)
- **Exhibit D** — The two drivers: salary growth + building projects
- **Exhibit E** — Fewer staff, more money: headcount down 9%, salaries up 34%
- **Exhibit F** — The structural mismatch: per-employee costs outpacing revenue
- **Exhibit G** — The levy rate option never pursued (current rate is less than half the statutory max)
- **Exhibit H** — Who gained, who lost: every executive kept their raise, every layoff was frontline

## Sources

All data comes from public records:

- TRL Final Budget documents ([trl.org/budget](https://www.trl.org/budget))
- Board meeting minutes and recordings
- BLS CPI-U data (Seattle-Tacoma-Bellevue, Series CUURS49DSA0)

For a detailed narrative account with full source citations, see the **[SAO Complaint Document](https://docs.google.com/document/d/199IHAeDQi0b-sb406NXkZzyeTm1rMX6hjIi6B_-IxRY/edit?usp=sharing)**.

## Technical Details

This is a single-file static site — no build tools, no dependencies. All charts are rendered with the HTML Canvas API. Just open `index.html` in a browser or visit the [live site](https://ecwilsonaz.github.io/trl-crisis-charts/).
